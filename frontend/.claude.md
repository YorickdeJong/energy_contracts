# Energy Contracts Design System

## Design Philosophy
Clean, minimal, and intuitive - inspired by Apple's design language.

## Colors

### Primary Colors
- **Primary Blue**: `#007AFF` - Primary actions, links, active states
- **Background White**: `#FFFFFF` - Main background
- **Secondary Background**: `#F5F5F7` - Cards, secondary surfaces
- **Border**: `#E5E5E7` - Borders, dividers

### Text Colors
- **Primary Text**: `#1D1D1F` - Headings, primary text
- **Secondary Text**: `#86868B` - Supporting text, labels
- **Tertiary Text**: `#C7C7CC` - Placeholders, disabled text

### Semantic Colors
- **Success**: `#34C759` - Success states
- **Warning**: `#FF9500` - Warning states
- **Error**: `#FF3B30` - Error states, destructive actions

## Typography

### Font Family
Use system font stack for native feel:
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif
```

### Font Sizes
- **Display**: text-4xl (36px) - Page titles
- **Heading**: text-2xl (24px) - Section headings
- **Subheading**: text-xl (20px) - Card titles
- **Body**: text-base (16px) - Body text
- **Caption**: text-sm (14px) - Captions, labels
- **Small**: text-xs (12px) - Tiny labels

### Font Weights
- **Semibold**: 600 - Headings, emphasis
- **Medium**: 500 - Buttons, labels
- **Regular**: 400 - Body text

## Border Radius

- **Buttons**: `rounded-xl` (12px)
- **Cards/Modals**: `rounded-2xl` (16px)
- **Inputs**: `rounded-lg` (8px)
- **Badges**: `rounded-full` (9999px)

## Spacing

Use 4px grid system:
- **xs**: 4px (1)
- **sm**: 8px (2)
- **md**: 16px (4)
- **lg**: 24px (6)
- **xl**: 32px (8)
- **2xl**: 48px (12)

## Shadows

- **Small**: `shadow-sm` - Subtle elevation
- **Medium**: `shadow-md` - Cards
- **Large**: `shadow-lg` - Modals, overlays

## Transitions

- **Duration**: 200ms - Smooth, responsive
- **Easing**: `ease-in-out` - Natural motion

## Component Guidelines

### Buttons
- Primary: Blue background, white text, xl rounded
- Secondary: Light gray background, dark text, xl rounded
- Ghost: Transparent, blue text, xl rounded
- Danger: Red background, white text, xl rounded

### Cards
- 2xl rounded corners
- White background
- Subtle shadow
- 24px padding

### Inputs
- lg rounded corners
- Border color: #E5E5E7
- Focus: Blue ring
- 12px padding

### Modals
- 2xl rounded corners
- Backdrop blur
- Slide-up animation
- Max width: 500px
